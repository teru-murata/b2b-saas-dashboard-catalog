# Public Repository Policy

This repository must be safe to share publicly.

Rules:
- Do not disclose named organizations, non-public links, non-public strategy, non-public review outputs, or non-public planning documents.
- Use generic BtoB SaaS and analytics dashboard language.
- Avoid overclaiming personal experience; show evidence through implemented UI and quality checks.
- Keep non-public source material and organization-specific matrices out of this repository.
- Treat raw prompts, model transcripts, and review outputs as private unless explicitly summarized and sanitized.
- Public presentation can adapt the same demo to different audiences, but repository text remains generic.
- Do not commit secrets, credentials, private data, customer data, raw logs, browser traces, or unnecessary local paths.
